# ひらがな れんしゅう 🌸

子ども向けの **ひらがな練習アプリ**（iPad対応のWebアプリ / PWA）。
指でなぞって書く練習をしながら、音声で発音を聞き、できたら楽しい演出でほめてくれます。

## ✨ 機能

- **あいうえお表** … 清音46文字。タップで練習開始。クリアした文字に⭐がつく（進捗は端末に自動保存）
- **なぞり書き** … Canvasに指でなぞって書く。お手本にどれだけ沿えたかを判定
- **成功演出** … 紙吹雪 🎉 ＋ 効果音 ＋ ほめ言葉でやる気アップ
- **音声読み上げ** … Web Speech API（日本語）で文字とことばを発音
- **ことば学習** … その文字で始まる単語＋イラスト（絵文字）を表示
- **PWA対応** … iPadのホーム画面に追加すると、全画面・オフラインで使える

## 🚀 使い方（ローカルで動かす）

ES Modules を使っているため、ファイルを直接開くのではなく簡易サーバー経由で開きます。

```bash
# このフォルダで実行
python3 -m http.server 8080
```

ブラウザで <http://localhost:8080> を開く。
iPadで使う場合は、同じWi-Fi内のPCのIPアドレス（例 `http://192.168.x.x:8080`）にアクセスし、
Safariの「共有」→「ホーム画面に追加」でアプリのように使えます。

## ☁️ 公開（ホスティング）

GitHub Pages で公開できます（HTTPS自動・PWA対応・無料）。
`main` への push で GitHub Actions が自動デプロイする構成が入っています。
初回セットアップ手順は [docs/HOSTING.md](docs/HOSTING.md) を参照。

## 📁 構成

```
index.html              画面のマークアップ
css/style.css           スタイル（子ども向けデザイン）
js/
  app.js                画面切り替え・進捗・成功演出のまとめ役
  data.js               ひらがな46文字＋ことばのデータ（ここを増やせば拡張できる）
  trace.js              なぞり書きの描画と判定
  speech.js             音声読み上げ（Web Speech API）
  audio.js              効果音（Web Audio APIで生成）
manifest.webmanifest    PWA設定
sw.js                   Service Worker（オフライン対応）
icons/icon.svg          アプリアイコン
```

## 🔧 文字やことばを増やすには

`js/data.js` の `HIRAGANA` にエントリを追加し、`GOJUON_ROWS` に表示位置を足すだけです。
（濁音・半濁音・拗音などは今後のIssueで対応予定）
